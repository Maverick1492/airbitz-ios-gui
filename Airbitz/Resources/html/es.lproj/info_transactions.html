Lista de transacciones

Cada transacción muestra un valor rojo o verde que es la cantidad que usted haya gastado o recibido en bitcoin y fiat (USD, CAD, EUR, etc.).

Pulse el botón izquierdo superior (Saldo) para ver su saldo corriente en cada transacción.

Toque en el saldo total en la parte superior para alternar entre mostrar su saldo u ocultarlo de la vista.

Buscar sus transacciones arrastrando hacia abajo en la lista de transacciones y utilizando el campo de búsqueda. Ingresar un beneficiario, la categoría o el texto de un campo de notas para buscar.

Pulse el encabezado de nombre de carpeta en la parte superior para listar sus otras carteras, crear una nueva carpeta, cambiar el nombre de una carpeta, eliminar una carpeta, exportar sus transacciones, o para seleccionar otra cartera para el uso.
