Solicitud de Bitcoin

Esta pantalla le permite solicitar bitcoin de otro usuario. Elija la cartera de destino haciendo clic en el nombre de la cartera en la cabecera superior.

Presente código QR al remitente y pídale que escaneee el código para enviar bitcoin. Una vez recibidos los bitcoins, esta pantalla automáticamente va a detectar la transacción y mostrará los detalles de la transacción, donde usted puede etiquetarla.

Pulse los botones [Email] o [SMS] para enviar una solicitud a un usuario remoto a través de correo electrónico o SMS.

También puede copiar la dirección de esta solicitud con el botón [Copy] y pegarlo en una cartera diferente para enviar fondos

Opcionalmente puede ingresar una cantidad a solicitar pulsando en el campo de texto [Escriba cantidad] o pulsando el código QR. Pulse el botón superior derecho a elegir entre bitcoin o moneda fiat (USD, EUR). Si no se especifica ninguna cantidad, el remitente determinará la cantidad de bitcoin para enviar.
