Exporter Info du portefeuille


Choisissez un portefeuille en utilisant le bouton du haut.

Sélectionner [CSV] pour exporter une liste CSV de toutes les transactions pour votre portefeuille sélectionné.

Sélectionner [Quickbooks] pour exporter un fichier de format QBO pour importer dans Quickbooks.

Sélectionner [Wallet Private Seed] pour exporter le capital privé du portefeuille sélectionné.
Sélectionner [Wallet Public Seed] pour exporter le capital publique (XPub) du portefeuille sélectionné.
ATTENTION, le capital privée donne accès à tous les fonds sur le portefeuille sélectionné. Garder le capital privée sécurisé et ne l’exposez pas à d'autres.
