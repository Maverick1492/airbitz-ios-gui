Экспорт информации о кошельке


Выберите кошелек для экспорта, используя кнопку наверху.

Выберите [CSV] для экспорта всех переводов выбранного кошелька в формате CSV.

Выберите [Quickbooks] для экспорта в формате QBO для использования в Quickbooks.

Выберите [Wallet Private Seed] для экспорта приватных сидов выбранного кошелька.
Выберите [Wallet Public Seed] для экспорта публичных сидов (XPub) выбранного кошелька.
ВНИМАНИЕ, приватные сиды дают доступ ко всем средствам выбранного кошелька. Храните приватные сиды надежно и не показывайте их никому.
