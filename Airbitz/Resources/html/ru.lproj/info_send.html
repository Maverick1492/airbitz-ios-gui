Сканирование и отправка биткойнов

Этот экран позволяет Вам отсканировать QR-код для отправки биткойнов или импортировать приватный ключ в Ваш кошелек.
Также Вы можете отсканировать QR-код BitID для входа на сайт, использующий BitID.
Выберите кошелек своей учетной записи, нажав на его название в верхнем заголовке.

Отсканируйте QR-код другого кошелька чтобы получить адрес кошелька для перевода средств.

Нажмите на кнопку [Photos] для выбора фото, содержащего QR-код, который Вы хотите отсканировать.

Нажмите [Address] для того, чтобы вставить или ввести вручную адрес получателя платежа.

Если Вы получите запрос на платеж по Bluetoooth, то он появится внизу. Нажмите на него, чтобы отправить деньги получателю.

Нажмите [Transfer] для перевода биткойнов в другой кошелек Вашей учетной записи. Просто выберите нужный кошелек во всплывающем окне.
