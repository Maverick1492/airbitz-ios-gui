Настройки

Используемый номинал биткойнов: Измените номинал, используемый для отображения биткойнов в приложении. (биты по умолчанию)

Учетная запись: Измените пароль своей учетной записи, PIN-код для снятия средств или вопросы и ответы для восстановления пароля.

Имя: По желанию Вы можете ввести свое имя, фамилию или псевдоним. Содержимое этого поля будет передаваться другим пользователям вместе с Вашими переводами. Эта информация никогда не будет доступна никому кроме лиц, с которыми Вы осуществляете переводы, ни [[abtag APP_TITLE]], ни любой третьей стороне.

Автоматический выход по завершении: Временной интервал, через который будет совершен автоматический выход из приложения после перевода его пользователем в фоновый режим.
Валюта по умолчанию: Задайте валюту по умолчанию, которая будет использоваться при создании новых кошельков и для отображения полного баланса в списке Ваших кошельков. Обратите внимание, что эта настройка не затрагивает валюту, используемую при переводах в отдельно взятых кошельках. Создавайте новые кошельки для работы с различными валютами.

Изменить категории: Настройте список категорий переводов, появляющийся в деталях перевода.

Пределы трат: Задайте дневной (защищенный паролем) и единократный (защищенный PIN-кодом) пределы трат в подтверждении перевода.

Двухфакторная аутентификация: Включите двухфакторную аутентификацию для повышения уровня защиты.

Режим магазина: Включите для перехода в удобный для ведения торговли режим переводов. После получения платежа приложение вернется к экрану запроса вместо предложения перейти к деталям перевода.

Bluetooth: Включите, чтобы иметь возможность запрашивать и отправлять запросы биткойнов лицам поблизости через Bluetooth Smart (BLE).

Повторный вход с использованием PIN-кода: Включите для использования для входа 4хзначного PIN-кода вместо полного пароля учетной записи.

Использовать TouchID: Включите для возможности входа по отпечатку пальца с использованием сканера TouchID на тех устройствах, где это поддерживается.

Обмен по умолчанию: Выберите сервис обмена, который определит кросс-курс биткойнов с различными валютами.
