Продавайте, храните, отправляйте и получайте биткойны

Простой в использовании биткойн-кошелек с широкими возможностями позволяет своим пользователям легко контролировать приватные ключи своих биткойнов и пользоваться ими так же легко и просто, как мобильным банковским клиентом.

Получите контроль над своими средствами! С Airbitz доступ к Вашим средства и информация о переводах будут только у ВАС. Ни Airbitz, ни какая-либо третья сторона не смогут получить доступ к Вашим средствам или данным. Вы получите тайну финансовых операций и автономность, для которых биткойны и создавались, с легкостью использования, доступной даже бабушкам.

Сфокусированный на предоставление пользователю потрясающего опыта, кошелек Airbitz предоставляет следующий набор великолепных функций:

• Нулевые знания и невозможность доступа к средствам пользователя, его ключам или данным о переводах как со стороны Airbitz, так и с любой третьей стороны
• Интегрированная купля-продажа для резидентов США и Канады
• Оплата с использованием Bluetooth (BLE) без помощи QR-кодов 
• Простое создание учетной записи, использующей только имя пользователя и пароль (не требует печати PDF-файлов, заполнения от руки кодовых фраз или настройки шифрования)
• Автоматическое создание зашифрованных резервных копий кошельков на выделенных пиринговых облачных серверах
• Двухфакторная аутентификация в одно касание - самое простое и изящное решение для защиты кошелька в мире
• Быстрое переключение между учетными записями через выпадающий список на экране входа
• Синхронизация между различными мобильными устройствами
• Иерархически определенные кошельки, изменяющие адрес для каждого перевода
• Учетная запись объединяет несколько кошельков с определяемыми пользователем названиями (например, "Деньги на отпуск")
• Поддержка нескольких валют. Вы можете задать свою валюту каждому кошельку
• Поиск переводов по получателю, категории или примечаниям
• Встроенный калькулятор для легкого расчета валютной конверсии
• Режим магазина предоставляет легкий и доступный пользовательский интерфейс для ведения торговли
• Определение частичных платежей позволяет создавать QR-коды для запросов на платеж с учетом оставшегося баланса 
• Дневные пределы трат (с паролем для их превышения)
• Пределы трат, контролируемые PIN-кодом
• Пользовательский интерфейс для быстрого перевода средств между кошельками в два клика
• Импорт средств через приватные ключи WIF или Casascius
• Децентрализованная архитектура сервера. Кошельки работают даже если сервера Airbitz лежат.
• Интеграция с деловыми каталогами Airbitz позволяет автоматически заполнять деловую информацию для переводов, включая логотипы и фото
• Интеграция со списком контактов позволяет автоматически заполнять имя получателя платежа и его фото
• Поддержка платежного протокола BIP 70
• Открытый исходный код. Доступен по адресу https://github.com/Airbitz

********* 	Уведомление о порядке обращения с личной информацией *********
Приложение Airbitz требует доступа к списку контактов на устройстве, геопозиционированию и личной информации. Эта информация используется, чтобы обеспечить пользователю наиболее удобные условия работы следующим образом:

1. Автозаполнение контактов из списка контактов пользователя после перевода
2. Автозаполнение контактов из адресной книги пользователя для отправления запросов на платеж по электронной почте или посредством SMS-сообщений
3. Автозаполнение деловых справочников после перевода
4. Использование геопозиционирования на устройстве пользователя для поиска близлежащих компаний

Ни личная информация, ни список контактов никогда не передаются без шифрования с использованием регистрационных данных пользователя (имя пользователя и пароль). Ни Airbitz, ни какая-либо третья сторона не имеют доступа к передаваемой приложением информации.
